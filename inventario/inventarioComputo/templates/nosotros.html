Acerca de nuestro proyecto

Este sistema de inventario fue creado por los alumnos Alonso Gutelius y Diego Linares como parte de un trabajo de investigación realizado como parte de los cursos de Proyectos I Proyectos II con fines educativos en el año 2023. Este proyecto fue realizado utilizando herramientas de computación en nube como AWS e Inteligencia Artificial haciendo uso de Python.

Aquí te mostramos 3 tecnologías que utilizamos para este proyecto:

Python

Ciencia de Datos

Se hizo uso de librerías de Machine Learning de Python como pandas, sci kit learn y plotly.

AWS

Servicios Web

Se utilizaron las herramientas de los servicios web de Amazon para poder desplegar la solución en la nube y garantizar la disponibilidad del sistema.

Django

Framework

Django es un marco de trabajo para aplicaciones web que permitió al sistema poder contar con herramientas para gestionar las peticiones HTTP y desarrollar el presente sistema web.
